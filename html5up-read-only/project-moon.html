Terraformed a Small Moon

Designing habitability from scratch taught me how resilience and empathy go hand in hand with every line of code.

The project began with a simple question: how do you adapt terrestrial ecosystems to a lunar environment? I led a cross-discipline team that built a predictive simulation, then progressively added data from real regolith and radiation experiments. Each iteration shaped a new capsule of documentation that could travel anywhere.

Problem / Goal

Create a self-sustaining lunar habitat model that balances thermal management, dust mitigation, and crew well-being.

Input / Output

Input: Regolith compaction, radiation measurements, greenhouse sensor streams.
Output: Predictive simulation and operational guide for fixed-duration missions.

Method

  • Couple thermal cycles, dust transport, and greenhouse controls inside a systems model.
  • Progressively inject field data from experiments to validate the simulations.
  • Build stakeholder dashboards that surface anomaly alerts.
  • Document workflows to preserve institutional knowledge for partner teams.

Result / Use

The model informed habitat design updates and became the basis for mission readiness briefings and automation scripts.

Skills / Tools

Systems modeling, Python, simulation pipelines, data storytelling, dashboarding.
