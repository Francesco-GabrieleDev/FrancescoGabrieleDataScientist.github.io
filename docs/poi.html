POI Geometries Retrieval with Image Segmentation from Satellite Imagery

Automated parcel extraction for spatial analytics.

Computer vision pipeline for point-of-interest geometry generation.

Problem / Goal

Accurate parcel geometries are foundational for spatial analytics, yet POI datasets often lack reliable polygon boundaries. This project automatically extracts high-quality polygon geometries for POIs directly from satellite imagery, reducing manual GIS effort and enabling scalable downstream analysis.

Input / Output

Input

  • High-resolution satellite images centered on POI locations.
  • Bounding-box annotations for POI structures.
  • Predefined geographic coordinates for each site.

Output

  • Polygon geometries representing POI parcels.
  • Georeferenced spatial footprints for GIS workflows and mobility analysis.

Method

  • Collected and curated a dataset of satellite images covering diverse POI environments.
  • Fine-tuned a YOLOv11-based segmentation model to detect and segment POI parcels.
  • Optimized training with data augmentation for robustness across urban/rural contexts.
  • Post-processed segmentation masks into clean polygon geometries.
  • Manually reviewed and corrected edge cases to ensure full spatial coverage and accuracy.

Result / Use

The model achieved reliable parcel segmentation across thousands of POIs, producing consistent, scalable geometry outputs. These polygons became the spatial foundation for downstream analyses (human activity aggregation, competition modeling, site performance studies), cutting manual GIS work while improving geometric consistency.
