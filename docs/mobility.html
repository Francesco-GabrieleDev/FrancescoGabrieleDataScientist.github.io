Fine-Grained Mobility Prediction for Retail Demand

Graph neural networks for site-level mobility forecasting.

Master’s thesis — applied to gas stations, with mobility used as a proxy for demand.

Problem / Goal

At store-level granularity, classical mobility models (e.g., gravity / OD flows) are often too coarse to explain or predict incoming activity. This project forecasts station-level human mobility on individual parcels and uses it as a proxy for retail demand, supporting performance benchmarking and planning decisions.

Input / Output

Input

  • Site polygons (parcel-level footprints) as spatial units.
  • Anonymized vehicle mobility trajectories (car-location pings).
  • Site features capturing parcel geometry, neighborhood context, and spatial competition.

Output

  • Predicted station-level mobility (incoming flux).
  • Demand proxy usable for site ranking, network planning, and scenario analysis.

Method

  • Mobility aggregation: counted trajectory endpoints and pings within each site polygon to estimate observed station-level mobility.
  • Feature engineering: built multi-scale predictors covering parcel/local context, street-network/urbanity descriptors, and spatial competition signals from nearby stations.
    • Parcel-level and local context features.
    • Street-network and urbanity descriptors.
    • Spatial competition signals from nearby stations.
  • Graph construction: connected stations via Delaunay triangulation with distance-decay edge weights to model local competitive structure.
  • Modeling: trained graph neural networks (GCN and GAT) to predict per-station mobility from node and edge information.
  • Evaluation: compared against non-graph baselines to quantify the value of spatial context and competition modeling.

Result / Use

Graph-based models captured competitive spillovers and spatial dependencies missed by non-graph approaches, improving predictive performance for fine-grained mobility.

  • Site-level performance comparison.
  • Network optimization and white-spot analysis.
  • Planning decisions where sales labels are missing or delayed.
